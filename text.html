Ahora, se definie cada variable y se indica el tipo de dato/variable con el que se trabajar, agrupandolas en dos grupos:

Categoricas/Cualitativas:

  • CustomerId : Identifiacin del cliente. Se transformar a factor. (factor/nominal).

  • Surname : Apellido de los clientes. Se transformar a factor. (factor/nominal).

  • Geography : Pais de origen de cada cliente. Se transformar a factor. (factor/nominal)

  • Gender : Genero de cada cliente. Se transformar a factor. (factor/nominal)

  • HasCrCard : Posee/No (1/0) posee Tarjeta de Credito. Se transformar a factor. (`factor``/nominal).

  • IsActiveMember : Indica si el cliente est/no est (1/0) activo en la entidad.

Todas se transformarn a factor. (factor/nominal).

Cuantitativas:

  • Age : Edad del cliente. (int/discreta).

  • Tenure : Meses faltantes para pago total de la deuda. (int/discreta)

  • CreditScore : Score de Credito, valoracin de cada cliente basado en el historial crediticio. (`int``/discreta).

  • EstimatedSalary : Salario estimado del cliente. (num/continua).

  • Balance : Deuda pendiente de pago. (num/continua).

  • NumOfProducts : Productos bancarios de cada cliente. (`int``discreta).

El valor CreditScore se usa para categorizarte en niveles que dependen del modelo de cada banco pero se suele usar:

  • 350 ~ 579 - poor

  • 580 ~ 669 - fair

  • 670 ~ 739 - good

  • 740 ~ 799 - very_good

  • > 800     - `excellent.`

Como puede convenir para el anlisis, se decide agregar variable categrica CreditCat (ord/ordinal). Adicional a los cambios de clase mencionados, se busca conocer si existen valores nulos o NA:

dat <-  data_ori[2:13] %>% 
  mutate( CreditCat= case_when(CreditScore < 580 ~ 1,
        CreditScore >= 580 & CreditScore < 669 ~ 2,
        CreditScore >= 669 & CreditScore < 739 ~ 3,
        CreditScore >= 739 & CreditScore < 799 ~ 4,
        CreditScore >= 799 ~ 5,
        TRUE ~ 0)) %>% 
  select(CustomerId, Surname,Geography,Gender, 
         IsActiveMember, HasCrCard, CreditCat,
         CreditScore, Age, EstimatedSalary, Tenure , 
         Balance, NumOfProducts) %>% 
  mutate(across(c(CustomerId,
                  Surname,
                  Geography,
                  Gender,
                  IsActiveMember,
                  HasCrCard,
                  CreditCat),
                factor)) %>% 
  rename_if(is.numeric,~paste("cu",.x,sep = "_") ) %>% 
  rename_if(negate(is.numeric),~paste("ca",.x,sep = "_") )
